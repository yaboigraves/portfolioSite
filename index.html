untitled rpg

- A long term passion project, an rpg about music and communities that flock around it. Set in a stagnant world where music is gatekept by tech giants.

- Combines elements of games and manga we love, like Ping Pong, Samurai Champloo, Jet Set Radio, Parappa the Rapper, Persona, Paper Mario, and more.

- Made with MFX indie collective (Jalen Hernandez (art), Jerry Joiner (design/writing), Aaron Patrick (3d), and yours truly (everything else plus all the other roles too).

Homebody

Get on Steam!

- A retro inspired psychological horror puzzler, about friendships and mental health. Created by Game Grumps Inc and Rogue Games.

- Time loop escape room, with a murderous killer stalking you.

- I worked as a generalist unity developer with a UI focus on this project.

- Very small (3 person core including me) team, so I wore alot of hats!
